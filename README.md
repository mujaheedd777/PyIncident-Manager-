## Description
A Python tool to help Blue Teamers log, track, and resolve security incidents. It saves all data locally using JSON so no reports are lost.

## Features
- [x] Add new IP incidents (Type, Severity)
- [x] View Open vs. Closed tickets
- [x] Persistent Storage (JSON)
- [x] Status Updates

## How to Run
1. Install Python 3
2. Run `python incident_tool.py`
